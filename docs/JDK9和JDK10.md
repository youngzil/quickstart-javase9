Java 9 尝鲜之JShell交互式编程环境
JShell/Kulla项目是第一个官方的Java REPL （Read-Eval-Print-Loop的缩写，即交互式编程环境），一种命令行工具。它允许你无需使用类或者方法包装来执行Java语句。它与Python的解释器类似，或其它本地支持REPL的JVM语言，如Scala和Groovy。
